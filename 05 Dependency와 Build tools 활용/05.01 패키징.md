# 패키징

라이브러리(Library)는 특정 기능들을 미리 구현해 놓은 코드와 자원들의 집합입니다. 개발자는 이 라이브러리를 자신의 프로젝트에 가져와 사용함으로써, 반복적인 코드 작성 시간을 절약하고 개발 효율성을 높일 수 있습니다.

## 실습 참고용 소스

- https://github.com/ohhoonim/practice-java-packaging.git


## Framework와 Library의 차이 

라이브러리는 개발자가 필요할 때 호출하여 사용하는 반면, 프레임워크(Framework)는 프로그램의 전체적인 구조(뼈대)를 제공하고 개발자가 그 틀에 맞춰 코드를 작성합니다

## 외부 라이브러리를 사용하는 방법

- 컴파일과 실행

컴파일 할 때 사용할 외부 라이브러리의 classpath을 명시 해주어야 합니다.

```sh
# -cp 옵션 뒤에 라이브러리 파일 경로를 지정
javac -cp "gson-2.10.1.jar" Main.java

# 클래스 경로에는 JAR 파일과 현재 디렉토리(.) 모두 포함해야 함
java -cp "gson-2.10.1.jar:." Main
```

- IDE (VSCode, IntelliJ, Eclipse 등)

외부 라이브러리를 사용하여 코딩 하는 도중에 코드가 정상적으로 작성되었는지 '정적분석' 기능을 사용하려면 IDE에서 별도의 설정이 필요로 합니다. 

## 빌드도구나 IDE없이 jar 컴파일 하기까지의 과정   

자바에서는 *.jar 확장형태로 라이브러리를 만들고 실행할 수 있습니다. 

1. 소스 구조
```sh
.
├── src/
│   ├── com/
│   │   └── myapp/
│   │       ├── Main.java     <-- 메인 클래스 (실행 진입점)
│   │       └── util/
│   │           └── Helper.java
└── lib/
    └── gson-2.10.1.jar       <-- 외부 라이브러리 (다운로드 완료 가정)
```

2. 컴파일 준비
```sh
# 컴파일된 .class 파일을 모아둘 임시 디렉토리(bin)를 생성합니다. 
$ mkdir bin

# -cp (Classpath) 옵션으로 gson.jar을 지정하고, 현재 디렉토리(.)도 포함
# -d (Destination) 옵션으로 컴파일된 클래스 파일을 bin 디렉토리에 저장
# Windows:
# javac -cp "lib\gson-2.10.1.jar;." -d bin src\com\myapp\Main.java src\com\myapp\util\Helper.java

# Linux/macOS:
$ javac -cp "lib/gson-2.10.1.jar:." -d bin $(find src -name "*.java")
```
💡 Tip: Unix 계열($() find)을 사용하면 모든 소스 파일을 자동으로 찾을 수 있어 편리합니다.

3. JAR 패키징 

실행 가능한 JAR를 만들기 위해 **매니페스트 파일(MANIFEST.MF)**이 필요합니다. 이 파일은 JAR의 메타데이터와 **실행 진입점(Main-Class)**을 JVM에 알려줍니다.

manifest.txt 파일을 생성하고 다음 내용을 입력합니다:
```txt
Main-Class: com.myapp.Main
Class-Path: lib/gson-2.10.1.jar
```
⚠️ 중요: Main-Class:와 Class-Path: 라인 끝에는 반드시 줄 바꿈(개행) 문자가 있어야 합니다. Class-Path 항목은 JAR 파일 내부가 아닌, 실행 시 JAR 파일을 기준으로 한 상대 경로를 명시합니다.

jar 명령어를 사용하여 bin 디렉토리의 모든 .class 파일과 manifest.txt를 묶어 MyApp.jar를 만듭니다.

```sh
# c: 새로운 JAR 파일 생성
# f: 파일 이름 지정 (MyApp.jar)
# m: 매니페스트 파일 지정 (manifest.txt)
# C: 디렉토리(bin)로 이동하여 내용물만 포함
$ jar -cfm MyApp.jar manifest.txt -C bin .
```

4. 최종 JAR 실행 

```sh
# 1. lib 디렉토리를 통째로 복사
cp -r lib .

# 2. 실행
java -jar MyApp.jar
```
