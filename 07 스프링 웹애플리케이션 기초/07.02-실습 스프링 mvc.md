### 실습 3) Jackson 라이브러리

- 스프링에서는 json 통신을 위해 Jackson 라이브러리를 사용합니다.
- Jackson라이브러리의 ObjectMapper 에 대해 알아보시오. 

```java
import com.fasterxml.jackson.databind.ObjectMapper;

// 1. 객체 직렬화 (Serialization)
ObjectMapper objectMapper = new ObjectMapper();
User user = new User("Charlie", 40);

// Java 객체를 JSON 문자열로 변환
String jsonString = objectMapper.writeValueAsString(user);
// 결과: {"name":"Charlie","age":40}

// 2. 객체 역직렬화 (Deserialization)
String inputJson = "{\"name\":\"David\",\"age\":50}";

// JSON 문자열을 User.class 객체로 변환
User deserializedUser = objectMapper.readValue(inputJson, User.class);
// 결과: deserializedUser.getName() -> "David"
```

### 실습 3) 예외 처리

- Spring MVC에서 예외를 처리하는 방법을 조사하시오. 
---
-  메소드내에서 직접 처리(try-catch) 
```java

@GetMapping("/item/{id}")
public ResponseEntity<Item> getItem(@PathVariable Long id) {
    try {
        Item item = itemService.findById(id);
        return ResponseEntity.ok(item);
    } catch (ItemNotFoundException e) {
        return ResponseEntity.status(HttpStatus.NOT_FOUND).build();
    }
}
```

- Controller 내에서 처리 (@ExceptionHandler)
 
```java
@RestController
@RequestMapping("/api/users")
public class UserController {

    // ... 핸들러 메소드 ...

    @ExceptionHandler(UserNotFoundException.class)
    @ResponseStatus(HttpStatus.NOT_FOUND) // 응답 상태 코드를 404로 설정
    public ErrorResponse handleUserNotFoundException(UserNotFoundException e) {
        // 예외 정보를 ErrorResponse DTO에 담아 반환
        return new ErrorResponse("USER_001", "사용자를 찾을 수 없습니다.");
    }
}
```

- 전역(Global) 예외 처리 (@ControllerAdvice)

```java
@RestControllerAdvice // 모든 Controller에 적용되며, @ResponseBody가 기본 적용됨
public class GlobalExceptionHandler {

    // 사용자 정의 예외 처리
    @ExceptionHandler(ItemNotFoundException.class)
    @ResponseStatus(HttpStatus.NOT_FOUND)
    public ErrorResponse handleItemNotFound(ItemNotFoundException e) {
        return new ErrorResponse("ITEM_NOT_FOUND", e.getMessage());
    }

    // 모든 예상치 못한 예외 처리 (최후의 방어선)
    @ExceptionHandler(Exception.class)
    @ResponseStatus(HttpStatus.INTERNAL_SERVER_ERROR)
    public ErrorResponse handleAllExceptions(Exception e) {
        return new ErrorResponse("SYSTEM_ERROR", "서버 오류가 발생했습니다.");
    }
}
```
