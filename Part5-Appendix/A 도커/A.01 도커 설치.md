# 도커 설치

## 도커란?

Docker는 애플리케이션과 그 종속성(라이브러리, 설정 파일 등)을 컨테이너(Container)라는 독립적이고 실행 가능한 패키지로 묶어주는 컨테이너화(Containerization) 기술입니다.

Docker를 사용하면 개발 환경, 테스트 환경, 운영 환경 등 어떤 환경에서든 애플리케이션을 일관되고 예측 가능한 방식으로 실행할 수 있습니다.

## Docker Desktop

- docker가 처음이라면 Docker Desktop을 다운로드하여 설치하기 바랍니다. 
- windows 사용자라면 Store에서 다운로드 받을 수도 있습니다.
- 설치 후 실행만 해두면 됩니다. 

## Spring Boot Docker Compose Support

- Dokcker Componse Support를 사용하면 spring boot 로 작성된 프로젝트를 개발하거나 서버 배포시 도커를 쉽게 사용할 수 있게 해줍니다.
