# JDBC

JDBC(Java Database Connectivity)는 Java 언어로 애플리케이션이 관계형 데이터베이스에 접속하여 데이터를 조회하고 조잘할 수 있도록하는 표준 Java API 입니다. 

스프링 프레임워크에서는 JDBC를 이용하기 보다는 조금더 편하게 사용할 수 있도록 몇가지 프레임워크를 제공해주고 있습니다. 여기서는 JDBC client의 사용법에 대해 알아봅니다.

## Fluent model의 사용 - JdbcClient

스프링 프레임워크 6.1부터 `NamedParameterJdbcTemplate` 과 `JdbcTemplate` 은 Fluent Interaction Model(유창한 상호작용 모델)을 갖춘 통합된 JdbcClient를 사용할 수 있게 되었습니다.

예를 들어, 위치 매개변수의 경우:

```java
private JdbcClient jdbcClient = JdbcClient.create(dataSource);

public int countOfActorsByFirstName(String firstName) {
	return this.jdbcClient.sql("select count(*) from t_actor where first_name = ?")
			.param(firstName)
			.query(Integer.class).single();
}
```

명명된 매개변수의 경우:

```java
private JdbcClient jdbcClient = JdbcClient.create(dataSource);

public int countOfActorsByFirstName(String firstName) {
	return this.jdbcClient.sql("select count(*) from t_actor where first_name = :firstName")
			.param("firstName", firstName)
			.query(Integer.class).single();
}
```

## RowMapper

`RowMapper`유연한 결과 해결 기능을 갖춘 기능도 제공됩니다.

```java
List<Actor> actors = this.jdbcClient.sql("select first_name, last_name from t_actor")
		.query((rs, rowNum) -> new Actor(rs.getString("first_name"), rs.getString("last_name")))
		.list();
```

사용자 정의 대신 `RowMapper` 로 매핑할 클래스를 지정할 수도 있습니다

```java
List<Actor> actors = this.jdbcClient.sql("select first_name, last_name from t_actor")
		.query(Actor.class)
		.list();
```

필수 단일 개체 결과: `single()`

```java
Actor actor = this.jdbcClient.sql("select first_name, last_name from t_actor where id = ?")
		.param(1212L)
		.query(Actor.class)
		.single();
```

`Optional`결과 는 다음과 같습니다: `optional()` 

```java
Optional<Actor> actor = this.jdbcClient.sql("select first_name, last_name from t_actor where id = ?")
		.param(1212L)
		.query(Actor.class)
		.optional();
```

## 업데이트

업데이트된 내용은 다음과 같습니다.

```java
this.jdbcClient.sql("insert into t_actor (first_name, last_name) values (?, ?)")
		.param("Leonor").param("Watling")
		.update();
```

또는 명명된 매개변수를 사용한 업데이트 문:

```java
this.jdbcClient.sql("insert into t_actor (first_name, last_name) values (:firstName, :lastName)")
		.param("firstName", "Leonor").param("lastName", "Watling")
		.update();
```

개별 명명된 매개변수 대신 매개변수 소스 객체를 지정할 수도 있다.
