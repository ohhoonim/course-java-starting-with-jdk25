* 컴퓨터공학과를 졸업하고, 현재 중소기업의 1년 차 Java 백엔드 개발자로 근무하고 있습니다.
* 회사에서 스프링 부트를 사용하여 기존 프로젝트의 유지보수 및 간단한 기능 추가 업무를 담당하고 있습니다.
