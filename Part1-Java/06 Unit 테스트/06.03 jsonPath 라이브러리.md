# jsonPath 라이브러리

JSON 문서(JSON document)에서 원하는 데이터를 쉽게 탐색(Query)하고 추출(Extract)하기 위해 고안된 표현식 언어입니다.

## JSON 이란?

JSON (JavaScript Object Notation)은 데이터를 교환하고 저장하기 위한 개방형 표준 파일 형식이자 데이터 교환 형식입니다.

사람이 읽고 쓰기 쉬우며, 컴퓨터가 파싱(Parsing)하고 생성하기도 쉽기 때문에, 현대 웹 환경에서 서버와 클라이언트 간에 데이터를 주고받는 표준 포맷으로 가장 널리 사용됩니다.

- 객체

    - 중괄호({ })로 감싸집니다.
    - 데이터는 **키(Key)**와 **값(Value)**의 쌍으로 구성되며, 콜론(:)으로 구분됩니다.
    - 키는 반드시 쌍따옴표("")로 묶인 문자열이어야 합니다.
    - 각 키-값 쌍은 쉼표(,)로 구분됩니다.
    - 프로그래밍 언어의 Map 또는 Dictionary와 유사합니다.

    ```json
    {
    "name": "홍길동",
    "age": 30,
    "is_student": false
    }
    ```

- 배열 

    - 대괄호([ ])로 감싸집니다.
    - 값들이 순서대로 나열되며, 쉼표(,)로 구분됩니다.
    - 프로그래밍 언어의 List 또는 Array와 유사합니다.

    ```json
    [
        "사과",
        "바나나",
        "딸기"
    ]
    ```

- 값 타입 

    JSON에서 값(Value)은 다음 6가지 데이터 타입 중 하나여야 합니다.

    - 문자열 (String): 쌍따옴표로 묶입니다. (예: "Hello")
    - 숫자 (Number): 정수 또는 부동 소수점 숫자. (예: 10, 3.14)
    - 객체 (Object): {}
    - 배열 (Array): []
    - 불리언 (Boolean): true 또는 false
    - Null: null

## JSONPath 기본 문법

| 구문            | 설명                                                                          | 예시                          | 결과                         |
| --------------- | ----------------------------------------------------------------------------- | ----------------------------- | ---------------------------- |
| $               | "JSON 문서의 **루트(Root                                                      | 최상위 객체)**를 나타냅니다." | $                            | 전체 JSON 객체 |
| .               | 하위 객체의 이름으로 접근합니다.                                              | $.store.book                  | store 객체 아래의 book 배열  |
| []              | 배열의 인덱스나 이름으로 접근합니다.                                          | $.book[0]                     | book 배열의 첫 번째 요소     |
| [*]             | 배열의 모든 요소를 선택합니다.                                                | $.book[*].title               | 모든 책의 제목               |
| ..              | 재귀적 탐색 (Deep Scan). JSON의 어느 위치에 있든 해당 이름의 필드를 찾습니다. | ..author                      | JSON 전체에서 모든 작가 필드 |
| [?(expression)] | 필터 표현식. 조건에 맞는 배열 요소만 선택합니다.                              | $.book[?(@.price < 10)]       | 가격이 10 미만인 책          |


## JSONPath 기본 사용법

```json
{
  "store": {
    "book": [
      { "category": "fiction", "author": "Herman Melville", "title": "Moby Dick", "price": 8.99 },
      { "category": "fiction", "author": "J. R. R. Tolkien", "title": "The Lord of the Rings", "price": 22.99 }
    ]
  }
}
```

```java
import com.jayway.jsonpath.JsonPath;

String json = "... 위 JSON 데이터 ...";

// 1. 모든 책의 제목(title)을 리스트로 추출
String path1 = "$.store.book[*].title";
List<String> titles = JsonPath.read(json, path1); 
// 결과: ["Moby Dick", "The Lord of the Rings"]

// 2. 가격이 10 미만인 책의 작가(author)를 추출
String path2 = "$.store.book[?(@.price < 10)].author";
List<String> authors = JsonPath.read(json, path2); 
// 결과: ["Herman Melville"]
```
