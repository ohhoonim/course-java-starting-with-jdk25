### 실습1) 버블 정렬 알고리즘

- 인접한 두 원소를 비교하여 정렬 순서에 맞지 않으면 교환(Swap)합니다

```java
void main(String[] args) {
    int[] nums = new int[] { 21, 4, 6, 39, 3, 19, 12 };
    bubbleSort(nums);
    System.out.println("버블 정렬 결과: " + Arrays.toString(nums));
    // 출력: [3, 4, 6, 12, 19, 21, 39]
}

void bubbleSort(int[] arr) {
    int n = arr.length;
    // 배열의 크기 - 1 만큼 반복 (이미 정렬된 마지막 원소는 제외)
    for (int i = 0; i < n - 1; i++) {
        // 정렬되지 않은 부분(뒤쪽)을 비교
        for (int j = 0; j < n - 1 - i; j++) {
            // 현재 원소가 다음 원소보다 크면 (오름차순) 교환
            if (arr[j] > arr[j + 1]) {
                // 교환 (Swap)
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
```

### 실습 2) 선택 정렬 알고리즘

- 전체 배열을 훑어 가장 작은(또는 큰) 값을 찾은 후, 
- 그 값을 정렬되지 않은 부분의 맨 앞 원소와 교환합니다. 
- 이 과정을 반복하여 정렬된 부분을 하나씩 늘려갑니다.

```java
void main(String[] args) {
    int[] nums = new int[] { 21, 4, 6, 39, 3, 19, 12 };
    selectionSort(nums);
    System.out.println("선택 정렬 결과: " + Arrays.toString(nums));
    // 출력: [3, 4, 6, 12, 19, 21, 39]
}

void selectionSort(int[] arr) {
    int n = arr.length;

    // i는 정렬될 위치 (맨 앞부터 하나씩 정렬된 원소를 확정)
    for (int i = 0; i < n - 1; i++) {
        // 현재 정렬되지 않은 부분에서 가장 작은 원소의 인덱스
        int minIndex = i;

        // i+1부터 끝까지 탐색하여 최소값 찾기
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }

        // 찾은 최소값(minIndex)과 현재 위치(i)의 원소를 교환
        // 교환 (Swap)
        int temp = arr[minIndex];
        arr[minIndex] = arr[i];
        arr[i] = temp;
    }
}
```

### 실습 3) 삽입 정렬 알고리즘

- 배열의 두 번째 원소부터 시작하여, 
- 현재 원소를 이미 정렬된 앞부분과 비교하며 자신의 올바른 위치를 찾아 삽입합니다

```java
void main(String[] args) {
    int[] nums = new int[] { 21, 4, 6, 39, 3, 19, 12 };
    insertionSort(nums);
    System.out.println("삽입 정렬 결과: " + Arrays.toString(nums));
    // 출력: [3, 4, 6, 12, 19, 21, 39]
}

void insertionSort(int[] arr) {
    int n = arr.length;

    // 두 번째 원소(인덱스 1)부터 시작하여 정렬된 부분(앞)에 삽입
    for (int i = 1; i < n; i++) {
        int key = arr[i]; // 현재 삽입할 원소
        int j = i - 1; // key의 왼쪽에 있는 정렬된 부분의 인덱스

        // 정렬된 부분(arr[j])을 key와 비교하여 key보다 큰 원소들을 오른쪽으로 한 칸씩 이동
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }

        // 올바른 위치에 key를 삽입
        arr[j + 1] = key;
    }
}
```

### 실습 4) 퀵 정렬 알고리즘 : 재귀

- 배열에서 피벗(Pivot) 원소를 하나 선택하고, 
- 피벗보다 작은 원소는 왼쪽으로, 큰 원소는 오른쪽으로 배치하여 배열을 둘로 나눕니다. 
- 이 과정을 재귀적으로 반복합니다.
- 재귀란 자기 자신을 직접 또는 간접적으로 호출하여 문제를 해결하는 프로그래밍 기법이다. 

> &nbsp;
> - (HINT) 수열의 점화식
> - 수열의 점화식은 초기조건(기저조건)과 점화관계(재귀 단계)로 표현한다
> - 점화 관계란 수열에서 현재 값이 나오려면 이전 값이 어떤 조건이어야 하는지를 표현한다
> 
> ``` 
> 예시) 피보나치 수열
> 	초기 조건(기저 조건) 
> 				F0 = 0
> 				F1 = 1
> 	점화 관계 (재귀 단계)
> 				Fn = F(n-1) + F(n-2) (단, n >= 2) 
> ```
> &nbsp;


```java
void main(String[] args) {
    int[] nums = new int[] { 21, 4, 6, 39, 3, 19, 12 };
    // 퀵 정렬은 전체 배열(0부터 끝까지)을 인수로 전달해야 합니다.
    quickSort(nums, 0, nums.length - 1);
    System.out.println("퀵 정렬 결과: " + Arrays.toString(nums));
    // 출력: [3, 4, 6, 12, 19, 21, 39]
}

// 퀵 정렬의 핵심: 분할(Partition) 함수
int partition(int[] arr, int low, int high) {
    // 배열의 가장 오른쪽 원소를 피벗으로 선택
    int pivot = arr[high];
    int i = (low - 1); // 피벗보다 작은 원소들의 끝 인덱스

    for (int j = low; j < high; j++) {
        // 현재 원소가 피벗보다 작거나 같으면
        if (arr[j] <= pivot) {
            i++;
            // i와 j의 원소를 교환
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }

    // 피벗을 i+1 위치로 이동 (피벗의 최종 위치)
    int temp = arr[i + 1];
    arr[i + 1] = arr[high];
    arr[high] = temp;

    return i + 1;
}

// 퀵 정렬 메인 함수 (재귀 호출)
void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        // 피벗을 기준으로 배열을 분할하고 피벗의 위치를 반환
        int pi = partition(arr, low, high);

        // 피벗을 제외하고 왼쪽과 오른쪽 부분을 재귀적으로 정렬
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```

### 실습 8) switch 가드 패턴

다음의 if문을 swith 가드 패턴으로 변환하시오.

```java
void main() {
   
   int score = 85;
   String grade;

   if (score >= 90) { // 90점 이상인 경우
       grade = "A";
   } else if (score >= 80) { // 90점 미만, 80점 이상인 경우
       grade = "B";
   } else if (score >= 70) { // 80점 미만, 70점 이상인 경우
       grade = "C";
   } else { // 70점 미만인 경우 (나머지)
       grade = "F";
   }

   IO.println("당신의 점수는 " + score + "점이며, 등급은 " + grade + "입니다.");
   
}
```

```java
// HINT)
void main() {
    int score = 85;

    // switch 표현식과 가드 패턴(when)을 사용하여 등급을 직접 변수에 할당합니다.
    String grade = switch (score) {
        // 90점 이상일 경우
        case int s when s >= 90 -> "A";
        
        // 80점 이상 90점 미만일 경우
        case int s when s >= 80 -> "B";
        
        // 70점 이상 80점 미만일 경우
        case int s when s >= 70 -> "C";
        
        // 나머지 모든 경우 (70점 미만)
        default -> "F"; 
    };

    IO.println("당신의 점수는 " + score + "점이며, 등급은 " + grade + "입니다.");
}
```
