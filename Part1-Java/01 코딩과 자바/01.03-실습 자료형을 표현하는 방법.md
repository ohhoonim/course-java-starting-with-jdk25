### 실습 7) 람다식 활용

람다식으로 함수형프로그래밍을 적용한다면 다음과 같이 메서드 파라미터로 ‘함수’를 제공하는 방법이 가능하다. 아래 예제를 참고하여 이것 저것 테스트해 보시오

```java
interface CalcAge {
	String calculate(int age, String type);
}

void main() {
  	CalcAge calcAge = (a, b) -> a + b;
	IO.println(new User().currentAge(calcAge, 10, "만"));
}

class User {
    private String name;
    private int age;

    public String currentAge(CalcAge calcAge, int age, String name) {
        return calcAge.calculate(age, name);
    }
}
```

- calcAge 의 람다식을 다음과 같이 변경해보자 

> (a, b) -> b + a + “ 세”
