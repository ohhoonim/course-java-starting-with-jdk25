### 실습 5) 삼항연산자 연습
주식 감정 평가액이 500원이다. 이것이 현재 주가 480원에 적정 평가 구간 -20 ~ +20 에 존재하는지를 검사하는 평가식을 작성하고, 평가구간내이라면 ‘적정’으로 출력하고 아니면 ‘평가불가’로 출력하시오
- 평가식은 논리 연산자와 비교 연산자로 작성할 수 있다.
- 삼항연산자를 이용하여 그 결과값을 변수에 저장할 수 있다.
hint)

```
void main() {
	int sum = 10 < 20 ? 20 - 10 : 10 - 20;
}
```
