# 01.01 Java 개발환경 구축

## 실습

### 실습 1) JDK 설치하기

- 가장 최신의 LTS 버전의 OpenJDK를 설치하시오.
- OpenJDK와 OracleJDK의 차이점을 조사하시오.

### 실습 2) VSCode 설치하고 첫 java 프로그램 작성하기

- VSCode를 설치하고 ‘_Extension Pack for Java_’ 확장을 설치하시오
- 작업공간으로 사용할 폴더를 하나 만들고 VSCode에 작업공간으로 추가하시오
- Main.java 파일을 하나 만들고 다음과 같이 코딩하시오

```java
void main() {
	IO.println(“자바로 코딩하기”);
}
```

- 작성된 자바코드를 실행하시오.

### 실습 3) Git 설치하기고 Github 가입하기

- git-scm.com 사이트로 이동하여 OS에 맞는 git 버전을 설치하시오
- github.com 사이트에서 회원가입을 하시오
