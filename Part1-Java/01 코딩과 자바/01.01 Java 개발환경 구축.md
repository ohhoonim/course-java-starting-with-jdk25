# 01.01 Java 개발환경 구축

소프트웨어는 OS라고 하는 운영체제에서 동작합니다. 소프트웨어를 개발 할 때에도 OS에 따라 개발 환경을 맞춰주어야 합니다. OS에는 대표적으로, 
- Windows
- MacOS
- Linux

이 세 가지로 구분이 됩니다. 본인의 OS가 뭔지 잘 모르겠다면 다음과 같이 체크해보세요

- 화면 상단에 메뉴가 있고 하단에 아이콘들이 위치해 있다. 애플에서 만든거다. 👉 MacOS
- 화면 하단에만 아이콘들이 있다. 유명한 대기업 브랜드이다. 👉 Windows

대부분의 경우 Lixux를 운영체제로 사용하는 컴퓨터를 사용하는 경우는 거의 없으므로 Linux 개발환경 구축에 대한 내용은 생략합니다

## JDK 설치하기

Windows에서는 가상의 Linux 환경을 사용할 수 있도록. WSL(Windows Subsystem for Linux)라는 것을 제공하고 있으나, 처음 배우는 경우에는 설치조차 어려울 수 있으니, Java개발에 익숙해지면 WSL을 이용하여 개발하는 방법을 시도해보기 바랍니다. 
여기서는 설치 프로그램 다운로드 방법으로 JDK를 설치해 보겠습니다.MacOS의 경우 brew 또는 sdkman 등을 터미널을 이용하여 설치하는 방법등도 있으나 마찬가지로 설치프로그램 다운로드 방식으로 진행합니다.  

> 📋 OpenJDK Temurin
> https://adoptium.net/temurin/releases/

다운로드 링크를 통해 웹 페이지에 접속한 후 <u>스크롤을 살짝 내려보면 자신의 OS에 맞는 다운로드 버튼을 클릭</u>하여 파일 다운로드 받는다. 탐색기(Windows) 또는 Finder(MacOS) 에서 다운로드 받은 파일을 찾아 더블 클릭(연속으로 두 번 클릭)하면 설치프로그램이 실행되고 화면에 보이는 순서에 따라 설치를 진행하면 됩니다. 설치 중 고민할 사항이 없으므로 설치 단계를 진행하여 JDK설치를 완료합니다. 

## 설치가 되었는지 확인하기

Windows에서는 Power Shell, MacOS에서는 Terminal 을 실행한 후 창에 커서가 깜박이면 다음과 같이 입력하여 설치 여부를 확인합니다. 아무런 내용도 나타나지 않는 경우 다음의 조치를 취한 후 다시 확인해 봅니다.
```sh
## 설치확인을 위한 명령어
$ java -version
```

- Power Shell 또는 Terminal을 재시작한 후 명령어를 입력해본다
- 재부팅 후 명령어를 입력해본다.
- 그래도 안되면 다시 설치를 해본다. 

## [나중에 시도해볼만한] SDKMAN으로 JDK 25 설치하기

sdkman은 소프트웨어 개발 도구들만 전문적으로 관리해주는 유용한 툴이다. 자바 개발환경에 익숙해지면 시도해보자. 

> 📋 SDKMAN 
> https://sdkman.io/

## Visual Studio Code 설치하기

Visual Studio Code 는 줄여서 VSCode라고 부릅니다. 아래 다운로드 링크로 이동하여 VSCode를 설치해도 되지만 Java 개발만 할 것이라면 ‘Getting Started’ 메뉴에서 Java 개발 전용 다운로드 파일을 제공해주고 있습니다.코딩이 처음이라면 자바 코딩 팩 다운로드를 추천합니다.

> 📋 Visual Studio Code 
> https://code.visualstudio.com/ 

> 📋 VSCode Java 개발 가이드 : 자바 코딩 팩
> https://code.visualstudio.com/docs/java/java-tutorial

VSCode의 다양한 확장 기능을 이용할 수 있겠으나 개발을 해보면서 필요한 것들을 하나씩 추가해나가면 됩니다. 단, 자바 코딩팩을 다운로드하지 않으신 분이라면 다음의 VSCode 확장을 필수로 설치해야합니다. 

- Extension Pack for Java

## 개발툴 비교 : VSCode vs IntelliJ

학습단계에서는 VSCode만으로 충분하다. 학생인 경우 IntelliJ를 무료로 사용할 수 있으나 그 외의 경우 Java 공부 이상으로 개발할 경우(스프링) 무조건 유료 결재를 할 수 밖에 없습니다. 

## 소스코드 버전관리 Git

소프트웨어의 가장 특징 중에 하나는 소스가 언제나 쉽게 변경이 된다는 것 입니다. 소스 변경을 관리하는 것을 ‘_형상관리_’라고도 합니다.대표적인 형상관리 툴로는 Git이라는 것이 있습니다.  Git을 이용할 수 있도록 해주는 대표적인 사이트는 다음과 같습니다.

- Github
- GitLab

GitHub은 가장 많은 개발자가 이용하고 있으며, Microsoft가 운영하는 곳 입니다. 아래 링크를 참고하여 Github 사이트 가입해 두시기 바랍니다. 

> 📋 Github
> https://github.com/

Github 가입과 별개로 git을 사용하려면 프로그램을 설치해야 합니다.

> 📋 Git
> https://git-scm.com/install/

코딩을 처음배우는 시점에는 그다지 사용할 일이 없지만 꾸준히 git 명령어를 연습해두어야 합니다.
