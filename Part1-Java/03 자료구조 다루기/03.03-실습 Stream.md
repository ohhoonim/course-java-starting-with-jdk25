### 실습 1) Collectors.groupBy

- Collectors.groupBy를 이용하여 부서별 직원을 그룹핑하는 코드를 작성하시오

```java
// 부서 Enum
enum Department {
    HR, DEVELOPMENT, SALES
}

// 직원 클래스
class Employee {
    String name;
    Department dept;
    int salary;

    Employee(String name, Department dept, int salary) {
        this.name = name;
        this.dept = dept;
        this.salary = salary;
    }

    @Override
    public String toString() {
        return name + " (" + salary + "만원)";
    }
}

void main() {
    IO.println("--- Collectors.groupingBy() 활용 예제 ---");

    // 1. 데이터 소스 (직원 목록)
    List<Employee> employees = List.of(new Employee("Alice", Department.DEVELOPMENT, 7000),
            new Employee("Bob", Department.HR, 5500),
            new Employee("Charlie", Department.DEVELOPMENT, 7500),
            new Employee("David", Department.SALES, 6000), new Employee("Eve", Department.HR, 6200),
            new Employee("Frank", Department.DEVELOPMENT, 8000));

    // 2. Stream 파이프라인 구성 (부서별 그룹화)
    // groupingBy의 결과 Map의 Key는 Department 타입, Value는 List<Employee> 타입이 됩니다.
    Map<Department, List<Employee>> employeesByDept = employees.stream()
            // groupingBy(그룹화 기준 함수)
            .collect(Collectors.groupingBy(emp -> emp.dept));

    IO.println("\n--- 부서별 그룹화 결과 ---");

    // 3. 결과 출력
    for (Map.Entry<Department, List<Employee>> entry : employeesByDept.entrySet()) {
        IO.println("부서: " + entry.getKey());
        IO.println("  직원: " + entry.getValue());
    }
}

```

### 실습 2) Gatherers 활용

- Gatherers.windowSliding() 을 사용하여 3일 이동평균을 구하는 로직을 구현하시오. 

```java

// List<Integer> 윈도우를 받아 평균(double)을 계산하는 헬퍼 함수
double calculateAverage(List<Integer> window) {
    if (window.isEmpty()) return 0.0;
    
    // 스트림을 이용해 윈도우 내 요소의 합계를 계산
    double sum = window.stream()
        .mapToInt(Integer::intValue) // IntStream으로 변환하여 sum() 사용
        .sum();
        
    // 합계를 요소 개수로 나누어 평균을 구합니다.
    return sum / window.size();
}

void main() {
    IO.println("--- Gatherers.windowSliding() 활용 예제 (이동 평균) ---");

    // 1. 데이터 소스 (일별 주가 데이터)
    // 7일간의 주가 데이터
    List<Integer> stockPrices = List.of(100, 105, 110, 108, 115, 120, 125);
    IO.println("원본 주가 (7일): " + stockPrices);

    int windowSize = 3;
    
    // 2. Stream 파이프라인 구성 (3일 슬라이딩 윈도우)
    List<Double> movingAverages = stockPrices.stream()
        // [Gatherer 연산] 요소를 3개씩 슬라이딩 윈도우로 묶습니다.
        .gather(Gatherers.windowSliding(windowSize))
        
        // [중간 연산: map] 각 윈도우(List<Integer>)를 받아 평균으로 변환합니다.
        .map(window -> calculateAverage(window))
        
        // [종단 연산: collect] 최종 결과를 List<Double>로 수집합니다.
        .collect(Collectors.toList()); 

    IO.println("\n--- 최종 결과 (3일 이동 평균) ---");
    IO.println("계산된 이동 평균: " + movingAverages);
    
    /*
    계산 과정:
    1. 윈도우 1: [100, 105, 110] -> 평균: 105.0
    2. 윈도우 2: [105, 110, 108] -> 평균: 107.66...
    3. 윈도우 3: [110, 108, 115] -> 평균: 111.0
    4. 윈도우 4: [108, 115, 120] -> 평균: 114.33...
    5. 윈도우 5: [115, 120, 125] -> 평균: 120.0
    */
}
```
