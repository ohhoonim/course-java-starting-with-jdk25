### 실습1 선택1) Windows 에서 PATH 설정하기

> 📋 Windows에서는 시스템 환경 변수 설정 창을 통해 Path를 설정합니다.
> - 제어판 또는 검색을 통해 "시스템 환경 변수 편집"을 실행합니다.
> - "환경 변수" 버튼을 클릭합니다.
> - "시스템 변수" 목록에서 Path 변수를 찾아 선택하고 "편집"을 클릭합니다.
> - 새로운 경로(예: C:\MyTool\bin)를 "새로 만들기"를 통해 추가하고 저장합니다.

### 실습1 선택2) Linux/Macos 에서 PATH 설정하기

> 📋  Linux나 macOS에서는 일반적으로 사용자 홈 디렉토리의 셸(Shell) 설정 파일(예: ~/.bashrc, ~/.zshrc, ~/.profile)을 수정하여 Path를 설정합니다.
> - 터미널을 열고 설정 파일을 편집기(vi 또는 nano)로 엽니다. (예: nano ~/.zshrc)
> - 파일의 맨 끝에 다음과 같이 새로운 경로를 추가하는 코드를 삽입합니다.
> - 주의: 기존 Path를 덮어쓰지 않도록 $PATH 변수를 함께 사용해야 합니다.
> 
> ```bash
> # Path에 /Users/user/Applications/MyTool/bin 경로 추가
> export PATH=$PATH:/Users/user/Applications/MyTool/bin
> ```
> 
> - 파일을 저장하고 닫습니다.
> - 변경된 Path 설정을 즉시 적용하기 위해 셸 설정 파일을 소싱합니다. (예: source ~/.zshrc)

### 실습 3) Reader와 Writer

- Reader와 Writer를 이용하여 파일 읽기, 쓰기 코드를 작성하시오 

```java
// 파일쓰기
void main() {
    String filename = "char_output.txt";
    String content = "안녕하세요. 이것은 자바 Reader/Writer 예제입니다.\n새로운 줄에 씁니다.";
    
    writeToFile(filename, content);
}

void writeToFile(String filename, String content) {
    IO.println("--- 파일 쓰기 시작 ---");
    
    // FileWriter를 try-with-resources로 선언하여 자동 닫기 보장
    try (FileWriter writer = new FileWriter(filename)) { 
        
        writer.write(content); // 문자열 데이터를 파일에 씁니다.
        writer.flush();        // 버퍼에 남아있는 데이터를 강제로 파일에 기록합니다.
        
        IO.println(filename + " 파일에 쓰기 완료.");
        
    } catch (IOException e) {
        IO.println("파일 쓰기 중 오류 발생: " + e.getMessage());
    }
}

```

```java
// 파일 읽기
public void readFromFile(String filename) {
    IO.println("\n--- 파일 읽기 시작 ---");
    
    // StringBuilder를 사용하여 읽은 문자열을 효율적으로 결합합니다.
    StringBuilder sb = new StringBuilder();
    int charCode; // 읽어온 문자 하나를 저장할 변수

    // FileReader를 try-with-resources로 선언하여 자동 닫기 보장
    try (FileReader reader = new FileReader(filename)) {
        
        // read()는 파일의 끝(-1)이 나올 때까지 반복하며 문자를 읽습니다.
        while ((charCode = reader.read()) != -1) {
            sb.append((char) charCode); // 읽은 정수(문자 코드)를 문자로 변환하여 추가
        }
        
        IO.println(filename + " 파일 읽기 완료.");
        IO.println("\n--- 파일 내용 ---");
        IO.println(sb.toString());

    } catch (IOException e) {
        IO.println("파일 읽기 중 오류 발생: " + e.getMessage());
    }
}

void main() {
    String filename = "char_output.txt";
    // 파일에 쓰기 (위의 writeToFile 메소드를 재사용한다고 가정)
    // new FileWriterExample().writeToFile(filename, "...");
    
    readFromFile(filename);
}
```
