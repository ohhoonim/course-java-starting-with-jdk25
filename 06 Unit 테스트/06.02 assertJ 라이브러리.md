# assertJ 라이브러리

AssertJ는 Java에서 유닛 테스트 코드를 작성할 때 사용되는 Fluent Assertion(유창한 단언) 라이브러리입니다. JUnit 자체에서도 assertEquals 같은 단언 메소드를 제공하지만, AssertJ는 훨씬 더 가독성 높고 표현력이 풍부한 테스트 코드를 작성할 수 있도록 돕습니다.

## Fulent API

AssertJ의 가장 큰 특징은 **메소드 체이닝(Method Chaining)**을 통해 영문 문장처럼 자연스러운 단언을 만들 수 있다는 것입니다.

- JUnit 스타일
 
    ```java
    assertTrue(list.size() == 3 && list.contains("Apple"));
    ```

- AssertJ 스타일

    ```java
    assertThat(list).hasSize(3).contains("Apple");
    ```

## AssertJ 작성 예시

```java
import org.junit.jupiter.api.Test;
// AssertJ의 핵심 클래스를 static import
import static org.assertj.core.api.Assertions.*; 

public class AssertJExampleTest {

    @Test
    void testStringAssertions() {
        // Arrange
        String actualMessage = "Welcome to AssertJ world!";

        // Assert (AssertJ 스타일)
        // 메소드 체이닝으로 여러 검증을 한 줄에 수행
        assertThat(actualMessage)
            .isNotEmpty() // 비어 있지 않은지
            .contains("AssertJ") // "AssertJ"를 포함하는지
            .doesNotContain("Goodbye") // "Goodbye"를 포함하지 않는지
            .isEqualToIgnoringCase("welcome to assertj world!"); // 대소문자 무시하고 같은지
    }
}
```
